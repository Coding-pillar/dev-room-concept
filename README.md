# Dealroom

A small Node.js social-media style app with a shorts-style scrolling feed, stories, post composer, likes, saves, comments, search, and infinite loading.

## Run

If Node.js is installed on your PATH:

```powershell
npm start
```

On this Codex workspace, the bundled Node runtime works even when `node` is not on PATH:

```powershell
& 'C:\Users\nechung\.cache\codex-runtimes\codex-primary-runtime\dependencies\node\bin\node.exe' server.js
```

Then open:

```text
http://localhost:3000
```
